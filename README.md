## ✦ Overview
This repository contains the source code for my personal portfolio website. Instead of relying on heavy frontend frameworks, this site is built from scratch using **HTML5, Custom CSS3, and Vanilla JavaScript** to maintain complete control over the DOM, animations, and typography. 

The design philosophy bridges deep technical engineering with a soft, pastel-themed UI (Blush Rose, Soft Lavender, Pastel Mint).

## ✦ Key Features
* **Meta-RAG Prompt Bar:** A custom JavaScript typing engine simulating a deterministic query against a personal latent space.
* **Theme Engine:** Fully custom CSS variables powering a seamless Light/Dark mode toggle that persists via LocalStorage.
* **3D Hardware Acceleration:** CSS `transform-style: preserve-3d` used for smooth directory card flips and hover states.
* **Serverless Architecture:** The contact form is routed through a Formspree asynchronous fetch pipeline, keeping the user on the page without backend infrastructure.
* **Responsive Bento Grids:** Modern CSS Grid layouts adapted for desktop, tablet, and mobile displays.
